## INSTALL
Get Phoenix and make the project, using:
```bash
git clone https://github.com/smortezah/Phoenix.git
cd Phoenix
cmake .
make
```


## RUN
```commandline
./phoenix [OPTION]... -m [MODELS] -r [REFERENCES] -t [TARGETS]
```


### OPTIONS
To see the possible options, type:
```bash
./phoenix -h
```
which provides the following:
```
SYNOPSIS


SAMPLE


DESCRIPTION


```
